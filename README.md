# Grep

一个简单实现 grep 的工具，支持正则表达式搜索和大小写不敏感匹配。

## 功能

- 支持正则表达式搜索
- 支持大小写不敏感匹配

## 使用

1. 基本搜索：
    ```sh
    cargo run -- "搜索关键词" 文件路径
    ```

2. 使用正则表达式搜索：
    ```sh
    cargo run -- -e "正则表达式" 文件路径
    ```

3. 忽略大小写搜索：
    ```sh
    IGNORE_CASE=1 cargo run -- "搜索关键词" 文件路径
    ```

## 测试

```sh
cargo test
```
